Sobre nós

Uma mercearia simples, acolhedora e presente no dia a dia do bairro

A Mercearia São José foi pensada para representar um comércio local tradicional, com foco em praticidade, confiança e atendimento próximo da comunidade.

Neste projeto acadêmico, a proposta é criar uma base visual sólida em HTML e CSS para depois evoluir o sistema com funcionalidades dinâmicas.
